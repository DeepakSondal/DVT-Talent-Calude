## 1. Scope of Pilot

DVT Talent AI grants Customer **free access** to the platform for the duration of the pilot period to:

- Run AI-powered talent sourcing pipelines (Discovery, Sourcing, Outreach agents)
- Manage candidate pipelines via the Naturalist Dashboard
- Use Copilot (Human-in-the-Loop) mode for manual review of candidates
- Sync jobs from connected ATS systems (Greenhouse / Ceipal, if applicable)

**Pilot credits allocated:** **10,000 AI credits** (sufficient for ~20 full pipeline runs)

**Excluded from pilot scope:**
- Custom integrations not listed above
- SLA guarantees (this is a pre-production evaluation)
- Multi-seat enterprise SSO (available in paid tiers)

- **Customer data** (job descriptions, candidate lists, API keys) is stored in an isolated tenant partition and is **never shared** with other customers.
- **Candidate PII** (emails, phone numbers) is encrypted at rest using AES-128 and in transit via TLS 1.3.
- Customer may request **full data deletion** at any time by emailing _________________.
- DVT will not use Customer's data to train AI models without explicit written consent.
- Data is hosted on _________________ (EU / US) in compliance with GDPR Article 28 (processor obligations).

## 4. Feedback Expectations

In exchange for free access, Customer agrees to:

- ✅ Complete the **Pilot Feedback Form** within 7 days of pilot end
- ✅ Participate in a **60-minute exit interview** with the DVT product team
- ✅ Share quantitative metrics (candidates sourced, time saved) using the Metrics Template
- ✅ Provide at least **one testimonial or case study quote** if the pilot is successful (optional but appreciated)

Customer is under **no obligation** to purchase DVT Talent AI after the pilot.
